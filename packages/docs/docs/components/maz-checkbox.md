## Basic usage

```v-model="{{checked}}"```

<MazCheckbox v-model="checked" :label="checked ? 'Checked' : 'Unchecked'" />

```vue
<template>
  <MazCheckbox v-model="checked" :label="checked ? 'Checked' : 'Unchecked'" />
</template>

<script lang="ts" setup>
  import { ref } from 'vue'
  import MazCheckbox from 'maz-ui/components/MazCheckbox'

  const checked = ref(false)
</script>
```
